DNS分离

centos7添加一块网卡，选择仅主机模式 第一块网卡配置ip 192.168.100.1作为局域网网关第二块网卡配置ip 12.0.0.1作为广域网网关客户机都选择仅主机模式给局域网客户机配ip给广域网客户机配ipvim /etc/named.conf进行配置

将监听的ip和允许的范围都改为any

将根域解析文件的内容注释掉或删除vim /etc/named.rfc1912.zones对/etc/named.rfc1912.zones进行配置，添加：
$$
view “lan” { match-clients { 192.168.100.0/24; }; zone “liren.com” IN { type master; file “liren.com.lan”; }; zone “.” IN { type hint; file “named.ca”; }; }; view “wan” { match-clients { 12.0.0.0/24; }; zone “liren.com” IN { type master; file “liren.com.wan”; }; };
$$
用“cp -p”命令保留权限复制一份“name.localhost”文件，命名为“liren.com.lan”。然后用vim编辑器对其进行编辑

更改文件配置
$$
$TTL 1D @ IN SOA liren.com. admin.liren.com. ( 0 ; serial 1D ; refresh 1H ; retry 1W ; expire 3H ) ; minimum NS liren.com. A 192.168.100.1 www IN A 192.168.100.99 dns IN A 192.168.88
$$
用“cp -p”命令保留权限复制一份“kgc.com.lan”文件，命名为“liren.com.wan”。然后用vim编辑器对其进行编辑输入：cp -p liren.com.lan liren.com.wan 输入：vim.liren.com.wan

更改文件配置
$$
$TTL 1D @ IN SOA liren.com. admin.liren.com. ( 0 ; serial 1D ; refresh 1H ; retry 1W ; expire 3H ) ; minimum NS liren.com. A 12.0.0.1 www IN A 12.0.0.1 dns IN A 12.0.0.1
$$
重启dns服务systemctl restart named分别使用两台客户机测试

都可以对域名进行解析
